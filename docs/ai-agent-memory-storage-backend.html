Why agent-memory Supports Multiple Backends

  • SQLite for speed — local, fast, no setup
  • Redis for scale — distributed, fast, production-ready
  • JSON for simplicity — file-based, no database needed
  • redis/agent-memory-server compatible — Redis backend for agent-memory
  • Google ADK compatible — ADK + Redis integration
  • MIT licensed — free for commercial memory backends
