The AI Agent Database NL2SQL Problem

AI agents can translate natural language into SQL queries and assist with database management. NL2SQL agents use RAG and multi-agent systems for accurate query generation. agent-memory stores schema context and query history.

cmcouto-silva/nl2sql-agent: RAG-Powered NL2SQL

"This project implements a session-based, RAG-powered AI agent that translates natural language into SQL queries for live querying of PostgreSQL." — cmcouto-silva/nl2sql-agent on GitHub

kbeaugrand/SemanticKernel.Agents.DatabaseAgent: Microsoft Semantic Kernel

"Powerful tool designed to generate SQL queries from natural language (NL2SQL) using Microsoft's Semantic Kernel framework. This project aims to bridge the gap between human-readable queries and SQL, enabling easy and efficient database interactions with AI-driven language models." — kbeaugrand/SemanticKernel.Agents.DatabaseAgent on GitHub

Yuvakunaal/AI-Powered-NL2SQL: Full-Stack OpenRouter + Mistral-7B

"NL2SQL Playground is a full-stack AI tool that converts natural language into SQL queries, allowing users to create tables, insert data, and run queries via a web UI or colorful CLI. Powered by OpenRouter + Mistral-7B, it provides step-by-step explainable query generation for fast, secure, and interactive database management." — Yuvakunaal/AI-Powered-NL2SQL on GitHub

Azure-Samples/azure-sql-nl2sql: Two-Experts Agent Model

"Two-Experts Agent Model to generate high-quality SQL queries from Natural Language request." — Azure-Samples/azure-sql-nl2sql on GitHub

Ali1858/llm_agents: Multi-Agent NL2SQL

"Demonstrate use case of multi and single LLM agents system for tasks like clinical metadata extraction and natural language to SQL." — Ali1858/llm_agents on GitHub
