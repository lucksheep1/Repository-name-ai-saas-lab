The OpenClaw Ecosystem

OpenClaw is an open-source AI agent framework that runs anywhere — any OS, any platform. The lobster way. Its ecosystem includes 5400+ skills, MCP tools, plugins, and deployment stacks. agent-memory is part of this ecosystem — a memory layer for OpenClaw agents with persistent context.
