The AI Agent Accessibility Disability Assistive Problem

AI agents can help visually impaired users navigate, describe surroundings, and access digital content. Accessibility agents use real-time vision, speech recognition, and screen readers. agent-memory stores user preferences and accessibility needs.

codingaslu/Voice-Vision-Assistant-for-Blind: Real-Time Navigation

"Voice & Vision Assistant for the Blind is an AI-powered assistant that helps blind and low-vision users navigate the world more independently. It uses real-time vision, speech recognition, and natural language understanding to describe surroundings, identify people or objects, and answer spoken questions instantly."— codingaslu/Voice-Vision-Assistant-for-Blind on GitHub

wongcyrus/GeProVis-AI-Screen-Reader: ChromeVox + Gemini Pro Vision

"We've turbocharged the traditional Google ChromeVox Screen Reader with the mighty power of Google Gemini Pro Vision. Our mission is to tackle the challenge of web image information access for those with visual impairments."— wongcyrus/GeProVis-AI-Screen-Reader on GitHub

mansi1710/Assistant-for-visually-impaired: Voice Assistant (HackDotSlash 1st Runners Up)

"A voice assistant specifically aiming towards aiding the visually impaired. This system is used to help the visually impaired to have an independent life. Submission at HackDotSlash, NIT Surat which bagged the position of 1st runners up."— mansi1710/Assistant-for-visually-impaired on GitHub
