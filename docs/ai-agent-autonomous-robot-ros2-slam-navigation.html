The AI Agent Autonomous Robot Problem

AI agents can now navigate autonomous robots through ROS2, SLAM, and motion planning algorithms. AI-powered autonomous robots use LiDAR, depth sensors, and Gazebo simulations to provide real-time obstacle avoidance, path planning, and warehouse navigation for mobile robots and delivery drones.

AI-Geniuses/Autonomous-Robot-Obstacle-Avoidance-with-ROS2

"This project focuses on developing an autonomous robot capable of real-time obstacle avoidance and navigation using ROS2. SLAM generates maps, while NAV2 handles path planning. Simulations are performed in Gazebo with RViz for visualization."— AI-Geniuses/Autonomous-Robot-Obstacle-Avoidance-with-ROS2 on GitHub

ai-winter/ros_motion_planning

"Motion planning and Navigation of AGV/AMR: ROS planner plugin implementation of A*, JPS, D*, LPA*, D* Lite, Theta*, RRT, RRT*, RRT-Connect, Informed RRT*, ACO, PSO, Voronoi, PID, LQR, MPC, DWA, APF, Pure Pursuit etc."— ai-winter/ros_motion_planning on GitHub

Lapland-Robotics/AI.R-Autonomous_Robot

"ROS2-based platform designed to streamline the development and deployment of autonomous mobile robots. At its core, it leverages ROS2 Navigation 2 for robust path planning and obstacle avoidance, integrates a flexible sensor suite."— Lapland-Robotics/AI.R-Autonomous_Robot on GitHub
