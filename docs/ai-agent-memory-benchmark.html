What No Benchmark Tests (and Why It Matters)

⚠ Benchmark Gaps in LoCoMo

LoCoMo tests memory accuracy — but doesn't measure:

• Encryption — Is conversation data encrypted at rest?
• TTL / Expiration — Does old memory auto-expire?
• MCP Protocol — Native MCP server support?
• Multi-agent sharing — Can multiple agents share a memory store?
• Self-hosting — Can you run it on your own infrastructure?
• EU AI Act compliance — Does data ever leave the device?

These dimensions determine real-world fit — especially for enterprise and privacy-sensitive deployments. A system that scores 90% but sends data to cloud servers may fail EU AI Act compliance requirements (August 2, 2026 deadline).
