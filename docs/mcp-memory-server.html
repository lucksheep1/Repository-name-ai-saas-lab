How agent-memory Works as an MCP Memory Server

  • MCP v3.2 native — connects to any MCP-compliant agent (Claude Code, Cursor, Cline, Windsurf, OpenClaw)
  • 5ms retrieval ready — JSON/SQLite backends optimized for fast reads
  • AES-256 encryption — only MCP memory server with mandatory encryption
  • TTL auto-expiration — stale memories auto-delete, no manual cleanup
  • Redis backend — shared memory across multiple MCP agents simultaneously
  • MIT license — deploy commercially, audit, modify without restrictions
