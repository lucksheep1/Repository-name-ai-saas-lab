## 正文

```
TL;DR: 做了个 150 行的 agent memory 库，无 LangChain 依赖。

背景：LangChain 很强大，但 100+ 包太大了。很多简单场景不需要。

方案：agent-memory
- ~150 行代码
- 2 个依赖 (pydantic, json)
- Pythonic API

pip install agent-memory

适合：简单 chatbot、agent 上下文、快速原型

欢迎试用反馈！
```
