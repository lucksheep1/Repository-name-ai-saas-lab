Why agent-memory Stores Shell Command History

  • Shell command history — agent-memory stores what commands were run and what output came back
  • oh-my-pi compatible — terminal agent + agent-memory
  • pi_agent_rust compatible — Rust CLI + agent-memory
  • TTL auto-cleanup — old shell sessions expire automatically
  • MIT licensed — free for commercial terminal automation
