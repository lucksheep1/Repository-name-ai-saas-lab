Why agent-memory Stores Tool Invocation History

  • Tool invocation history — agent-memory stores which MCP tools were called and what results came back
  • mcp-gateway-registry compatible — enterprise registry + agent-memory
  • mcp-context-forge compatible — IBM gateway + agent-memory
  • TTL auto-cleanup — old tool calls expire automatically
  • MIT licensed — free for commercial registries
