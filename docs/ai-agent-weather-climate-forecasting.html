The AI Agent Weather Climate Forecasting Problem

AI agents can predict climate variables, optimize agricultural planning, and monitor carbon emissions. Weather and climate agents use ML models for high-resolution forecasting. agent-memory stores historical weather patterns and environmental data.

climatechange-ai-tutorials/climatelearn: ML Climate Prediction

"Apply machine learning to predict climate variables into the future and transform low-resolution outputs of climate models into high-resolution regional forecasts."— climatelearn on GitHub

Orpheus-AI/Zeus: Bittensor Environmental Forecasting

"The Zeus Subnet leverages advanced AI models within the Bittensor network to forecast environmental data. This platform incentivizes innovation in climate science by enabling miners and validators to develop and evolve efficient, decentralized prediction models."— Orpheus-AI/Zeus on GitHub

SavTechSavvy/SmartWeather-agent-ai: Azure + OpenAI Weather Agent

"The goal of this app is to showcase an AI agent that combines the capabilities of Azure AI, OpenAI, and weather forecasting to provide intelligent weather insights."— SavTechSavvy/SmartWeather-agent-ai on GitHub

KKulma/climate-change-data: Climate Change Data Curated List

"A curated list of APIs, open data and ML/AI projects on climate change. Data and 96+ hour forecast of carbon intensity and generation mix for each country/region/postcode in the UK. This is the free data store of the EU..."— KKulma/climate-change-data on GitHub
