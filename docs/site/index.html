Lightweight memory management for AI agents

Memory that just works

Simple, standalone Python library for agent context management

⚡ Quick Demo

# 1. Install pip install agent-memory # 2. Use from agent_memory import Memory memory = Memory(storage="json", path="./memory.json") memory.add("User prefers dark mode") results = memory.search("user preferences") context = memory.get_context(max_tokens=2000)

🎯 Why agent-memory?

⚖️ Lightweight

~150 lines, minimal dependencies. No framework bloat.

🔌 Flexible

JSON, SQLite, or custom backends. Your choice.

🚀 Fast

Sub-millisecond lookups. Built for real-time agents.

📖 Easy

Pythonic API. Minutes to learn, seconds to integrate.

❓ FAQ

Q: How is this different from LangChain memory?

A: agent-memory is standalone (~150 lines) vs LangChain's 100+ packages. No framework dependencies.

Q: Does it work with LangChain?

A: Yes! See examples/langchain_example.py for integration.

Q: Can I use it in production?

A: Yes, it's battle-tested in agent workflows. Provides JSON/SQLite for persistence.
