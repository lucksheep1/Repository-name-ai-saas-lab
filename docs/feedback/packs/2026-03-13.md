## 项目概述

agent-memory 是一个轻量级 AI Agent 记忆管理库，专为需要管理对话上下文和长期记忆的 AI 应用设计。

**核心特点**:
- 轻量级：无框架依赖
- 简单易用：Pythonic API
- 多存储后端：JSON、SQLite
- 标签与优先级管理

---

## 痛点（来自 Evidence）

1. **LangChain 过于复杂**: 100+ 版本，依赖框架，Memory 模块文档冗长
2. **轻量级替代需求**: 社区讨论中多次提及需要更简单的 memory 解决方案
3. **Agent 上下文管理痛点**: 如何高效管理多轮对话上下文
