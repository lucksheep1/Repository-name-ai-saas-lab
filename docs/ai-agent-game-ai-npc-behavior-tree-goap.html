The AI Agent Game AI Problem

AI agents can now plan NPC behavior, generate game levels, and create roguelike games through GOAP and deep reinforcement learning. AI-powered game AI uses Goal-Oriented Action Planning, A* pathfinding, and behavior trees to provide intelligent NPC decision-making, procedural content generation, and dungeon generation for game developers.

caesuric/mountain-goap

"A GOAP (Goal Oriented Action Planning) AI library, written in C#. GOAP is a type of an AI system for games popularized by the F.E.A.R. AI paper. GOAP agents use A* pathfinding to plan paths through a series of actions."— caesuric/mountain-goap on GitHub

luxkun/ReGoap

"Generic C# GOAP (Goal Oriented Action Planning) library with Unity3d examples. GOAP is a decision-making technique for agents (NPCs, workers, enemies, simulation entities) where behavior is generated from world facts instead of hard-coded state machines."— luxkun/ReGoap on GitHub

FreddyWordingham/goap

"Goap-AI a library for creating AI agents which plan their actions based on a set of overall goals."— FreddyWordingham/goap on GitHub

SestoAle/DeepCrawl

"Deep reinforcement learning for the development of RogueLike games. We designed, built, and implemented a new RogueLike game with a working Deep Reinforcement Learning model."— SestoAle/DeepCrawl on GitHub
