The AI Agent 3D Modeling Rendering Problem

AI agents can automate 3D modeling, generate CAD scripts, and control Blender through natural language. LLM-powered 3D agents use function calling to execute commands in Blender. agent-memory stores project context and modeling patterns.

FreedomIntelligence/BlenderLLM: LLM Generates CAD Scripts

"BlenderLLM: A LLM specifically designed to generate CAD scripts based on user instructions. These scripts are then executed in Blender to render 3D models."— FreedomIntelligence/BlenderLLM on GitHub

BlenderAI/BlenderRL: AI Learns Blender Controls

"An open-source project for Blender 3D connects all the controls within Blender to an AI, allowing the AI to learn how to use the program through trial-and-error. This results in an efficient AI assistant for Blender, allowing for the smooth creation of new models, improvement of existing ones, automatic material generation, and optimized workflows."— BlenderAI/BlenderRL on GitHub

3D-Agent-Live/3D-Agent-Blender-AI-Plugin-PUBLIC: Blender AI Plugin

"https://3D-Agent.com is a Blender AI Plugin and Tool to Make Your Life in Blender Easier."— 3D-Agent-Live/3D-Agent-Blender-AI-Plugin-PUBLIC on GitHub

saofund/LLM-Blender-Agent: Claude Function Calling

"This is a project that uses the Function Call capability of different Large Language Models (LLMs) to operate Blender. The project also enhances workflow efficiency by integrating intelligent command execution and automation."— saofund/LLM-Blender-Agent on GitHub
