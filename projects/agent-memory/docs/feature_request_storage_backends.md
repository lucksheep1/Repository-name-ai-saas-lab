# Feature Request: Additional Storage Backends

## Summary
Request for adding more storage backend options to Agent Memory v3.x

## Proposed Backends
- **Redis** - For distributed systems and caching
- **PostgreSQL** - For relational data needs
- **MongoDB** - For document storage
- **S3** - For blob storage

## Priority
1. Redis (highest demand based on community feedback)
2. PostgreSQL
3. MongoDB

## Use Cases
- Multi-agent systems needing shared memory
- Enterprise deployments requiring SQL
- Large-scale document storage
