# Agent Memory v3.1 Release Notes

**Released:** 2026-03-21

---

## What's New in v3.1

### ✨ Human-Readable String TTL

Previously, TTL required integer days. Now use natural formats:

```python
from agent_memory import Memory

# Default TTL for all memories
m = Memory(ttl="7d")

# Per-item TTL
m.add("session data", ttl="1h")
m.add("temp cache", ttl="30m")
m.add("weekly backup", ttl="2w")
m.add("debug trace", ttl="30s")
```

**Supported formats:** `30s` (seconds), `5m` (minutes), `1h` (hours), `7d` (days), `2w` (weeks)

### 🔒 Encryption Support

Protect sensitive data with Fernet AES encryption:

```python
m = Memory(encryption_key="your-password-or-base64-key")

# Encrypt a specific memory
m.add("api_key_xxx", encrypt=True)

# Automatic decryption on retrieval
data = m.get(memory_id)
print(data["text"])  # Decrypted automatically
```

Key derivation: Password → PBKDF2-HMAC-SHA256 → 32-byte Fernet key.

### 🗄️ Redis Backend (Beta)

Multi-agent memory with Redis:

```python
m = Memory(
    storage="redis",
    redis_url="redis://localhost:6379",
    ttl="30m"
)
```

### 🔍 TTL Query API

Check remaining time on any memory:

```python
remaining_seconds = m.ttl_remaining(memory_id)
# Returns: 3542.5 (seconds), or None if no expiry
```

### 🐛 Bug Fixes

- **Sentinel value fix:** `ttl=None` now explicitly disables TTL (vs unset which uses default)
- **Graceful fallbacks:** Works without optional dependencies (Redis, FAISS, cryptography)

## Migration from v3.0

```python
# v3.0
m = Memory(ttl_days=7)
m.add("data", ttl_days=1)

# v3.1 (backward compatible)
m = Memory(ttl="7d")       # NEW: human-friendly
m = Memory(ttl=7)         # OLD: still works
m.add("data", ttl="1h")   # NEW: string format
m.add("data", ttl=1)      # OLD: still works
m.add("data", ttl=None)   # NEW: explicitly no TTL
```
