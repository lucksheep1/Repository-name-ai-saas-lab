# Agent Memory v3.2 - MCP Server

**MCP (Model Context Protocol) Server for Agent Memory**

Expose your agent memory as MCP tools for Claude Code, Cursor, Windsurf, and other AI IDEs.

## What's New in v3.2

- ✅ MCP Server - JSON-RPC based server
- ✅ 6 Tools exposed via MCP
- ✅ Works with any MCP-compatible client

## Installation

```bash
pip install agent-memory
```

## Quick Start

```python
# Run MCP server
python -m agent_memory.mcp_server

# Or import and use programmatically
from agent_memory.mcp_server import AgentMemoryMCPServer
```

## MCP Tools

| Tool | Description | Parameters |
|------|-------------|------------|
| `memory_add` | Add a memory | `text`, `ttl`, `tags`, `encrypt` |
| `memory_search` | Search memories | `query`, `top_k` |
| `memory_get_recent` | Get recent | `limit` |
| `memory_get_by_tag` | Filter by tag | `tag` |
| `memory_get_context` | Get context | `max_tokens`, `max_memories` |
| `memory_stats` | Statistics | (none) |
