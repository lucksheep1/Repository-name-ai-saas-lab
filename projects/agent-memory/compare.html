When to Choose What

Use Case Best Choice
Need cloud-managed service, no self-host Mem0 — $24M funded, production-grade
Temporal-aware pipelines, production Zep — time-aware memory, good docs
Long-running agents, framework user Letta — agent framework with memory
Want MCP-compatible local memory agent-memory ✓ — only MCP-native option
Privacy-first, no cloud dependency agent-memory ✓ — JSON/SQLite on your machine
Need Redis backend integration agent-memory ✓ — built-in Redis support
Why agent-memory? It's the only open-source AI agent memory library with native MCP v3.2 support. Run locally with JSON, scale to production with Redis — no cloud lock-in. Built for AI agents that need persistent memory across sessions.

Quick Start

# Install
pip install agent-memory

# Use with MCP (Cursor, Claude Code, OpenClaw)
python -m agent_memory.mcp_server

# Or use directly in Python
from agent_memory import Memory
m = Memory(storage="json", path="./memory.json")
m.add("User prefers dark mode", ttl="30d")
results = m.search("dark mode preference")


# MCP Tools available
memory_search  — Search memories
memory_add    — Add new memory
memory_get     — Get memory by ID
memory_list    — List recent memories
memory_clear   — Clear memories
