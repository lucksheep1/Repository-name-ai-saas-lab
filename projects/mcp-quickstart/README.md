# MCP Quick-Start v2

一键生成 MCP Server 模板的 CLI 工具 (入门版)。

## Problem

MCP (Model Context Protocol) 生态正在爆发，但开发者不知道如何快速创建自己的 MCP server。

## Solution

的入门模板提供简洁，帮助初学者快速上手：

- `basic` - 最简模板 (~30 行)
- `hello` - Hello World 示例

## Usage

```bash
cd projects/mcp-quickstart
python3 mcp-quickstart.py basic
```

## 包含内容

- 标准化的项目结构
- 注释详细的示例代码
- 基础的 package.json

## 与 MCP Templates 的区别

| 特点 | MCP Quick-Start | MCP Templates |
|------|-----------------|---------------|
| 目标 | 初学者入门 | 生产级模板 |
| 复杂度 | 最低 | 多种场景 |
| 代码量 | ~30 行 | ~80-150 行 |
| 语言 | Node.js | Python |
