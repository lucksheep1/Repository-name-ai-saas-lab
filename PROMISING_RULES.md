# Promising 评级规则 (v2.0)

**生效时间:** 2026-03-07

## 规则

### 1. Promising 名额
- **上限: 2 个**
- 其余项目降级为 Experiment（除非满足硬条件）

### 2. Promising 硬条件（缺一不可）
- [ ] README 含 Quick Demo（复制命令即可看到结果）
- [ ] VERIFICATION.md 含可复制步骤 + 预期输出
- [ ] Opportunity/Evidence 至少 3 条可追溯来源（issue/review/thread 链接或编号）

### 3. Analyst 评分
- 新增 Evidence Quality (0-10)
- **<7 不得 Promising**

### 4. 48h 深挖规则
- 只允许深挖 **1 个主项目**（证据最硬、迭代最连续）
- 其余只做 bugfix/文档，不加功能
