# Feedback Collection Snippets

## GitHub Issue 模板

### Bug Report
```markdown
## Bug Description
<!-- 简短描述问题 -->

## Steps to Reproduce
1. 
2. 
3. 

## Expected Behavior
<!-- 期望行为 -->

## Actual Behavior
<!-- 实际行为 -->

## Environment
- OS: 
- Python version: 
- Library version: 

## Additional Context
<!-- 截图/日志 -->
```

### Feature Request
```markdown
## Feature Description
<!-- 你想要什么功能? -->

## Use Case
<!-- 解决什么问题? -->

## Proposed Solution
<!-- 建议的实现方式 -->

## Alternatives Considered
<!-- 考虑过的替代方案 -->
```

### General Feedback
```markdown
## Feedback Type
- [ ] Bug
- [ ] Feature Request
- [ ] Question
- [ ] Other

## Your Use Case
<!-- 你怎么使用这个项目? -->

## What Worked Well
<!-- 什么做得好的? -->

## What Could Be Improved
<!-- 什么可以改进? -->
```

## Survey Questions (SURVEY.md)

```markdown
# 用户调查

## 1. 你如何得知这个项目?
- [ ] GitHub Search
- [ ] Twitter/HN
- [ ] 朋友推荐
- [ ] 其他

## 2. 你的主要使用场景是什么?
- Chatbot 记忆
- Agent 上下文管理
- 其他

## 3. 你愿意为哪些功能付费?
- 云端同步
- 更多存储后端
- 企业支持
- 其他

## 4. 你觉得最需要改进的是什么?
- [ ] 文档
- [ ] 示例
- [ ] 性能
- [ ] 功能

## 5. 你愿意参与贡献吗?
- [ ] 是，我会提交 PR
- [ ] 是，我会报告 Bug
- [ ] 否，我只是用户
```
