# Skills Registry

## 目录

### 1. Feishu (飞书)
- `skills/feishu/` - 飞书文档、云空间、知识库操作
- 包含：doc, drive, wiki, bitable, chat 操作手册

### 2. External Search (外部检索)
- `skills/external_search/` - Web search, fetch, browser automation
- 包含：搜索、爬取、浏览器控制

### 3. Project Management (项目管理)
- `skills/project_management/` - 项目创建、迭代、验证流程
- 包含：MVP 构建、Scale Gate、反馈收集

### 4. GitOps
- `skills/gitops/` - Git 提交规范、仓库管理

### 5. Research (研究)
- `skills/research_packs/` - 外部检索记录

---

## 使用规则

1. 遇到新问题 → 先查 skills/ 是否已有
2 → 附验证. 无则新增步骤（可运行命令/预期输出）
3. 每轮 Scout/Scanner → 必须输出 Research Pack
