# archive-ingest

文档统一归档技能。当用户发送触发指令时激活，按《文档录入标准 v1》执行。

---

## 适用输入

| 类型 | 描述 |
|------|------|
| 纯文字 | 消息正文直接作为内容 |
| 链接 | HTTP/HTTPS URL |
| PDF | PDF 文件 |
| 截图/图片 | 截图或图片附件（png/jpg/gif/webp） |

---

## 归档标准 vs 当前默认落点

```
归档标准（必须遵循）：定义了"什么必须写入、什么必须校验"
  ↓
当前项目默认落点：~/.openclaw/workspace/archive/
  总索引文档：~/.openclaw/workspace/archive/index.md
  归档文档：~/.openclaw/workspace/archive/[作者或来源]/[YYYY-MM-DD].md
```

skill 本身不硬编码落点路径；落点由项目配置文件或调用方指定。

---

## 执行流程

### Step 1：识别输入类型

| 类型 | 判断依据 |
|------|----------|
| 纯文字 | 消息正文即为内容 |
| 链接 | 消息含 URL |
| PDF | 收到文件路径或文件上传 |
| 截图/图片 | 收到媒体附件 |

### Step 2：内容提取

| 类型 | 提取方式 |
|------|----------|
| 纯文字 | 直接使用消息正文 |
| 链接 | web_fetch 抓取页面文本 |
| PDF | PDF 解析提取文本 |
| 截图/图片 | image 工具 → 保留原始识别文本输出（不经摘要加工） |

### Step 3：判断一级目录

```
[已存在的一级目录]：直接使用
archive/
  [作者或来源]/
    YYYY-MM-DD.md

[不存在的一级目录]：必须先向用户确认，未经确认不得自行创建
  archive/
    [待确认]/
      YYYY-MM-DD.md   ← 用户确认前不得创建真实一级目录
```

### Step 4：写入归档文档

**内容层必须保留**，不能以摘要替代。

截图/图片：原始识别文本完整粘贴，不做删减。

```markdown
# [归档日期] [输入类型] [标题/来源]

**来源：** [发布者 / URL / 文件名]
**归档日期：** YYYY-MM-DD HH:MM
**输入类型：** 纯文字 / 链接 / PDF / 截图
**归档路径：** [当前项目落点]/[作者或来源]/[YYYY-MM-DD].md

---

## 原文 / 原始识别文本

[全文粘贴，不做删减。不允许以"见摘要"代替。]
[若无法获取：填写"未获取 / 识别失败 + 原因"]

---

## 摘要

[有则填，无则填"无"。禁止留空。]

---

## 复核标记

- read-back 通过：✅ / ❌
- 原文/OCR 首句：[实际读取的前50字，逐字copy]
- 异常说明：[有则填，无则填"无"]

---
*录入完成时间: YYYY-MM-DD HH:MM*
```

### Step 5：read-back 复核（必须）

必须从文件系统重新读取归档文件，验证：
1. 内容层非空
2. 返回内容与预期一致

read-back 必须提供"原文/OCR 首句"作为可观察证据，不得在 read-back 之前报告"完成"。

### Step 6：更新总索引

总索引追加本次归档的路径和基本信息。

---

## 停止与确认条件

以下情况必须先向用户确认，不得自行推进：

1. 需要新增一级目录（作者/来源目录不存在）时
2. 内容层为空，无法提取原文/OCR 文本时
3. 链接 404 或权限限制，无法获取页面内容时
4. read-back 失败，文件内容与预期不符时

---

## 回报字段（完成后必须提供）

```
✅ 文档录入完成

归档路径：archive/[作者或来源]/[YYYY-MM-DD].md
总索引：index.md（已更新 / 未更新）
输入类型：[纯文字/链接/PDF/截图]
作者/来源：[实际作者名 或 来源名 或 待确认]
内容层：✅ 已写入（共 N 字）/ ❌ 缺失
原文/OCR 首句：[实际读取的前50字，逐字copy，不做任何转述]
复核：✅ read-back 通过 / ❌ 未通过
摘要：✅ 已写入 / 无
⚠️ 异常说明：[无]
```

---

## 禁止行为

- 不得在 read-back 之前报告"完成"
- 不得以摘要替代内容层
- 不得未经确认自行创建新一级目录
- 不得跳过 image 工具原始输出直接写摘要
