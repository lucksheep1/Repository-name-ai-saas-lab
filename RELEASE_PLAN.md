# Release & Promotion Workflow

## Release Checklist

- [ ] README 完整 (至少包含: 安装、使用、示例)
- [ ] 代码验证通过 (VERIFICATION.md)
- [ ] LICENSE 文件 (默认 MIT)
- [ ] 版本标签 (v1.0.0)
- [ ] GitHub Repo 创建/推送

## 推广计划

### 1. 发布当天
- [ ] Twitter/X 发帖 (简洁 + Demo GIF)
- [ ] GitHub Discussions 开启

### 2. 第一周
- [ ] Hacker News Submit
- [ ] Reddit (r/opensource, r/AI, r/programming)
- [ ] LinkedIn (技术社区)

### 3. 持续收集反馈
- [ ] GitHub Issues 模板
- [ ] README 添加反馈链接
- [ ] 定期回复社区评论

## 目标指标
- ⭐ Day 1: 10 stars
- ⭐ Week 1: 50 stars
- ⭐ Month 1: 100 stars
